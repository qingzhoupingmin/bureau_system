# 天津市市政工程局综合管理系统 - 打包部署指南

## 一、打包方式选择

本系统提供三种打包方式，根据需求选择：

### 1. 文件夹版本（推荐）✅
- **文件名**：`快速打包.bat`
- **优点**：启动快、体积小、稳定
- **缺点**：需要复制整个文件夹
- **适用场景**：首次部署、内部使用

### 2. 单文件版本
- **文件名**：`打包.bat`
- **优点**：只有一个 exe 文件
- **缺点**：启动慢、体积大
- **适用场景**：分发到多台电脑

### 3. 自定义配置
- **文件名**：`build.py`
- **优点**：完全自定义配置
- **缺点**：需要手动配置
- **适用场景**：专业用户

## 二、快速开始

### 最简单的打包步骤：

1. **双击运行 `快速打包.bat`**
   - 会自动检查环境
   - 自动安装依赖
   - 自动完成打包

2. **打包完成后**
   - 找到 `dist\天津市市政工程局综合管理系统\` 文件夹
   - 将整个文件夹复制到目标电脑
   - 双击 `天津市市政工程局综合管理系统.exe` 运行

## 四、部署到其他电脑

### 1. 文件夹版本部署

**在本机打包后：**
```
复制整个文件夹：
dist\天津市市政工程局综合管理系统\n```

**到目标电脑：**
- 将文件夹粘贴到任意位置
- 运行 `天津市市政工程局综合管理系统.exe`

### 2. 单文件版本部署

**在本机打包后：**
```
复制单个文件：
dist\天津市市政工程局综合管理系统.exe
```

**到目标电脑：**
- 将文件粘贴到任意位置
- 双击运行

### 3. 配置数据库连接（重要！）

**场景 A：使用本机数据库**
- 无需修改配置

**场景 B：使用远程数据库**
- 在目标电脑上创建 `config.py` 文件：
```python
DB_CONFIG = {

    'port': 3306,
    'user': 'root',
    'password': 'Oppor831t',
    'database': 'tianjin_bureau',
    'charset': 'utf8mb4'
}
```

## 五、常见问题解决

### 问题 1：打包失败

**原因**：缺少 PyInstaller
```bash
# 解决方法
pip install pyinstaller
```

### 问题 2：运行时找不到模块

**原因**：某些模块未包含在打包中
```bash
# 解决方法：修改打包脚本，添加 hidden-import
--hidden-import="模块名"
```

### 问题 3：启动很慢

**原因**：单文件版本需要解压
```bash
# 解决方法：使用文件夹版本
双击：快速打包.bat
```

### 问题 4：无法连接数据库

**原因**：配置文件问题或数据库不可访问
```bash
# 解决方法：
1. 检查数据库是否启动
2. 检查网络连接
3. 检查 config.py 配置
4. 检查防火墙设置
```

### 问题 5：防病毒软件误报

**原因**：PyInstaller 打包的exe容易被误报
```bash
# 解决方法：
1. 添加到白名单
2. 使用数字签名（需要证书）
3. 通知用户添加信任
```

## 六、高级配置

### 添加程序图标

1. 准备 `.ico` 格式图标文件（建议 256x256）
2. 放置到 `assets\images\icon.ico`
3. 重新打包

### 添加版本信息

1. 创建 `version_info.txt` 文件
2. 在打包脚本中引用
3. 查看详细说明参考 `打包说明.md`

### 创建安装程序

1. 下载并安装 Inno Setup
2. 使用提供的 `setup.iss` 脚本
3. 生成安装程序
