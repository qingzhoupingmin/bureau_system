# 天津市市政工程局综合管理系统 - 打包说明

## 打包前的准备

### 1. 安装必要工具

```bash
# 安装 PyInstaller
pip install pyinstaller
```

### 2. 安装依赖包

```bash
pip install -r requirements.txt
```

## 快速打包

### 方法一：使用打包脚本（推荐）

```bash
python build.py
```

### 方法二：直接使用 PyInstaller

```bash
pyinstaller --clean --noconfirm main.spec
```

### 方法三：不使用 spec 文件（简单打包）

```bash
pyinstaller --onefile --windowed --name "天津市市政工程局综合管理系统" main.py
```

## 打包文件说明

### main.spec（配置文件）
- 定义打包规则和包含的文件
- 可以自定义图标、版本信息等
- 包含所有必要的模块和资源

### build.py（打包脚本）
- 自动化打包流程
- 检查依赖和必要文件
- 提供友好的打包过程提示

## 打包后的文件

### dist/ 目录
- **天津市市政工程局综合管理系统.exe** - 主程序（单文件版本）
- 或 **天津市市政工程局综合管理系统/** 目录（文件夹版本，包含所有依赖）

### build/ 目录
- 打包过程中的临时文件
- 可以删除以节省空间

## 自定义配置

### 添加图标
1. 准备 .ico 格式图标文件（建议 256x256）
2. 放置到 `assets/images/icon.ico`
3. 重新打包

### 添加版本信息
在 main.spec 中修改：

```python
exe = EXE(
    ...
    version='version_info.txt',  # 版本信息文件
    ...
)
```

## 常见问题

### 1. 打包后无法启动
- 检查是否包含所有必要的文件
- 查看是否有 .log 文件记录错误信息
- 尝试使用 `--console` 参数查看控制台输出

### 2. 缺少模块
在 main.spec 的 `hiddenimports` 中添加：
```python
hiddenimports=[
    'pymysql',
    '缺少的模块名',
],
```

### 3. 文件过大
- 使用 `--onedir` 模式（文件较多但启动快）
- 排除不必要的模块
- 使用 UPX 压缩

### 4. 防病毒软件误报
- 这是 PyInstaller 打包程序的常见问题
- 可以添加数字签名以降低误报率
- 将程序添加到白名单

## 更新打包

```bash
# 清理旧的打包文件
pyinstaller --clean --noconfirm main.spec
```

## 技术支持

如遇到问题，请检查：
1. Python 版本是否兼容（推荐 Python 3.8+）
2. PyInstaller 版本是否最新
3. 所有依赖是否正确安装
4. 是否有文件权限问题
