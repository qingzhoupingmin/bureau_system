# 天津市市政工程局综合管理系统 - 使用文档

## 一、系统概述

**系统名称**: 天津市市政工程局综合管理系统

**技术栈**: Python 3.8+ + MySQL 8.0 + Tkinter

**功能定位**: 面向天津市市政工程局机关及下属单位的综合管理平台，涵盖资产管理、公文流转、科研经费管理、预算管理等功能。系统采用三级管理体系：局机关 → 中层单位（公路处、高速处、道桥处等）→ 基层单位。

---

## 二、系统环境要求

### 2.1 运行环境
- Python 3.8 或更高版本
- MySQL 8.0 或更高版本
- Windows 7/10/11

### 2.2 依赖安装
```bash
cd tianjin_bureau_system
pip install -r requirements.txt
```

### 2.3 数据库配置
编辑 `config.py` 文件中的数据库配置：
```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',
    'password': 'Oppor831t',
    'database': 'tianjin_bureau',
    'charset': 'utf8mb4'
}
```

### 2.4 初始化数据库
首次使用前，需要初始化数据库：
```bash
python run_init.py
```
这将自动创建数据库和所有必要的表，并初始化默认数据。

### 2.5 启动系统
```bash
python main.py
```

## 三、各角色界面说明

系统根据不同角色显示不同的功能界面，登录后默认显示"单位概况"。

### 3.1 局领导界面
导航菜单：单位概况 → 综合仪表盘 → 待办审批 → 公文查看 → 消息通知 → 统计分析 → 个人中心

功能说明：
- **综合仪表盘**：查看资产总数、预算总额、科研项目等统计卡片，以及待办事项列表
- **待办审批**：集中处理公文、预算、科研审批
- **公文查看**：查看所有公文
- **消息通知**：查看所有消息
- **统计分析**：资产、预算、科研、公文、用户统计报表
- **个人中心**：个人信息、修改密码

### 3.2 资产管理处界面
导航菜单：单位概况 → 资产管理 → 申请审批 → 资产统计 → 用户管理 → 个人中心

功能说明：
- **资产管理**：添加、编辑、删除资产（显示全系统所有资产）
- **申请审批**：批准/拒绝资产申请
- **资产统计**：按类别、状态、处室统计
- **用户管理**：添加、编辑、删除用户

### 3.3 办公室界面
导航菜单：单位概况 → 公文管理 → 消息通知 → 业务请示 → 个人中心

功能说明：
- **公文管理**：新建、编辑、发布公文
- **消息通知**：发送局名义/业务通知
- **业务请示**：查看并回复下属单位的请示

### 3.4 科技处界面
导航菜单：单位概况 → 科研项目管理 → 科研经费管理 → 申请审批 → 个人中心

功能说明：
- **科研项目管理**：查看所有科研项目
- **科研经费管理**：查看经费详情、拨付经费
- **申请审批**：批准/拒绝科研项目

### 3.5 财务处界面
导航菜单：单位概况 → 预算审批 → 预算分配 → 预算统计 → 个人中心

功能说明：
- **预算审批**：批准/拒绝预算申请
- **预算分配**：查看已批准预算
- **预算统计**：按年度统计预算情况

### 3.6 各处室界面（专业处室）
适用于：计划处、设施处、养护处、建设处、规划处、规费处、审计处、劳动人事处、安保处、法规处、党委办、纪检委、宣传部、组织部、老干部处

导航菜单：单位概况 → 部门资产 → 资产申请 → 消息通知 → 专业功能 → 个人中心

功能说明：
- **部门资产**：查看本处室资产
- **资产申请**：申请使用公用设备
- **消息通知**：接收通知
- **专业功能**：显示处室专属功能入口

**特殊功能**：
- 劳动人事处具有"账号注册"功能，可为下属单位或新员工创建账号
- 劳动人事处具有"密码修改审批"功能，可审批用户的密码修改申请

### 3.7 中层单位界面
适用于：公路处、高速处、道桥处、研究院、设计院、建设公司、地铁处

导航菜单：单位概况 → 本单位及下属单位资产 → 资产申请 → 公文提交 → 收文查看 → 草稿管理 → 预算申报 → 个人中心

功能说明：
- **单位概况**：查看本单位职能介绍及下属单位列表
- **本单位及下属单位资产**：查看本单位及所有下属单位的资产
- **资产申请**：申请使用公用设备
- **公文提交**：向局机关/平级单位/下属单位提交公文（先保存为草稿）
- **收文查看**：查看已发布的收到的公文
- **草稿管理**：管理本单位及下属单位提交的草稿，可编辑、发布、删除
- **预算申报**：可代下属单位提交预算申请，也可自己申报

### 3.8 基层单位界面
适用于：各中层单位下属的基层单位（如一桥所、收费中心等）

导航菜单：单位概况 → 单位资产 → 资产申请 → 公文提交 → 收文查看 → 草稿管理 → 预算申报 → 个人中心

功能说明：
- **单位资产**：查看本单位资产
- **资产申请**：申请使用公用设备
- **公文提交**：向所属中层单位提交公文（先保存为草稿）
- **收文查看**：查看已发布的收到的公文
- **草稿管理**：管理本单位提交的草稿，可编辑、发布、删除
- **预算申报**：预算申报由上级单位（所属中层单位）统一管理

## 四、功能模块详细说明

### 4.1 单位概况
- **本单位介绍**：查看本单位职能详细介绍
- **下属单位列表**：中层单位可查看下属单位列表
- **机关处室**：19个机关处室列表（网格布局，每行3个），点击可查看详细介绍

### 4.2 资产管理
- **资产管理处**：可查看和管理全系统所有资产
- **各处室**：可查看本部门资产
- **中层单位**：可查看所有下属单位资产
- **下属单位**：可查看本单位资产
- 资产添加时需选择归属单位（从下拉框选择）
- 资产类别需从下拉框选择（办公设备、办公家具、公用设备、消耗品）

### 4.3 公文管理（办公室）
- 新建公文：创建新公文
- 公文审核：审核待发布公文
- 发布公文：将公文发布

### 4.4 消息通知（办公室）
- 发送通知：向各处室/下属单位发送通知
- 局名义通知：使用局名义发布重要通知

### 4.5 业务请示（办公室/各处室）
- 查看请示：查看下属单位提交的请示
- 回复请示：回复业务请示

### 4.6 公文提交与草稿管理（下属单位/中层单位）
- **公文提交流程**：
  1. 进入"公文提交"模块
  2. 选择收文单位（上级/平级/下级）
  3. 填写标题、类型（报告/请示/通知）、内容
  4. 可选择上传附件
  5. 点击"提交公文"保存为草稿

- **草稿管理**：
  - 查看本单位及下属单位（中层单位）的所有草稿
  - **新建草稿**：直接创建新草稿
  - **编辑**：修改选中的草稿内容
  - **发布定稿**：将草稿发布为定稿，发布后接收方可在"收文查看"中看到
  - **删除**：删除不需要的草稿

- **收文查看**：
  - 查看已发布的收到的公文（只显示定稿）
  - 显示发送单位、发送时间、状态等信息

**公文流转流程**：
```
创建公文(草稿) → 草稿管理 → 发布定稿 → 接收方收文查看
```

### 4.7 预算申报
- **中层单位**：可为下属单位代为申报预算，也可自己申报
- **基层单位**：预算申报由上级单位统一管理，需联系上级单位办理

### 4.8 科研管理（科技处）
- 项目列表：查看所有科研项目
- 项目审批：批准/拒绝科研项目
- 经费管理：拨付科研经费

### 4.9 账号注册（劳动人事处）
- 可为下属单位或新员工创建账号
- 需填写用户名、姓名、初始密码
- 需选择归属单位和角色

### 4.10 个人中心
- 个人信息：查看用户名、姓名、角色
- 修改密码：提交密码修改申请（需劳动人事处审批）

## 五、三级管理体系说明

本系统采用三级管理体系：

### 第一级：局机关
- 19个处室
- 负责全局政策制定、审批管理

### 第二级：中层单位（7个）
| 单位 | 下属基层单位数 | 功能权限 |
|-----|---------------|---------|
| 天津市公路处 | 5个 | 管理下属单位资产、科研、预算 |
| 天津市高速公路管理处 | 12个 | 管理下属单位资产、科研、预算 |
| 天津市道路桥梁管理处 | 10个 | 管理下属单位资产、科研、预算 |
| 天津市市政工程研究院 | 7个 | 管理下属单位资产、科研、预算 |
| 天津市市政工程设计研究院 | 1个 | 管理下属单位资产、科研、预算 |
| 天津市市政工程建设公司 | 3个 | 管理下属单位资产、科研、预算 |
| 天津市地铁管理处 | 1个 | 管理下属单位资产、科研、预算 |

### 第三级：基层单位
- 归属于各中层单位
- 业务功能（科研申报、预算申报等）由所属中层单位统一管理

## 六、API 接口服务（可选）

### 6.1 启动 API 服务

```bash
# 安装依赖
pip install fastapi uvicorn pydantic

# 启动服务
cd tianjin_bureau_system
python -m uvicorn api.main:app --reload --port 8000
```

## 七、常见问题

### Q1: 忘记密码怎么办？
A: 联系系统管理员重置密码，或使用admin账号登录后通过用户管理重置。

### Q2: 如何修改个人信息？
A: 进入"个人中心"模块修改。

### Q3: 如何提交资产申请？
A: 在"资产申请"模块中选择设备，填写申请原因后提交。

### Q4: 下属单位如何提交公文？
A: 登录下属单位账号，进入"公文提交"模块，填写标题、内容、类型后提交。

### Q5: 基层单位如何申报预算？
A: 基层单位的预算申报由所属中层单位统一管理，请联系上级单位办理。

### Q6: 中层单位如何查看下属单位资产？
A: 登录中层单位账号，进入"下属单位资产"模块查看。

### Q7: 如何申请修改密码？
A: 进入"个人中心"模块，填写原密码、新密码、确认密码后提交申请。申请将提交至劳动人事处审批，审批通过后密码才会生效。

### Q8: 劳动人事处如何审批密码修改？
A: 登录劳动人事处账号（如劳动人事处1/劳动人事处123），进入"专业功能"模块，使用"密码修改审批"功能批准或拒绝申请。

### Q9: 劳动人事处如何注册新账号？
A: 登录劳动人事处账号，进入"账号注册"模块，填写用户名、姓名、密码，选择归属单位和角色后创建。

### Q10: 系统无法启动怎么办？
A: 检查以下几点：
1. MySQL数据库是否已启动
2. 数据库配置是否正确（config.py）
3. 依赖包是否已安装（pymysql, Pillow, cryptography）
