# 桌面应用启动指南

## 快速启动（2步走）

### 第1步：安装依赖
```bash
cd d:\测试开发项目\电商系统企业级+自动化框架\tianjin_bureau_system

```

## 详细启动步骤

### 1. 环境准备

#### 1.1 Python版本要求
- Python 3.8 或更高版本

#### 1.2 检查Python版本
```bash
python --version
```

#### 1.3 Tkinter要求
- Tkinter是Python的标准库，通常已安装
- 如果没有安装，可以使用：
```bash
pip install tk
```

### 2. 数据库准备

#### 2.1 启动MySQL服务
```bash
# Windows
net start MySQL80
```

#### 2.2 检查数据库连接
```python
python -c "from db.connection import db; print('数据库连接成功')"
```

#### 2.3 初始化数据库（如果需要）
```bash
python db/init_db.py
```

### 3. 安装依赖

#### 3.1 安装requirements.txt中的依赖
```bash
pip install -r requirements.txt
```

#### 3.2 依赖包列表
```
pymysql>=1.1.0        # MySQL数据库驱动
pyinstaller>=5.0.0      # 打包工具
requests>=2.28.0       # HTTP请求库
fastapi>=0.100.0       # API框架（可选）
uvicorn>=0.23.0        # ASGI服务器（可选）
pydantic>=2.0.0        # 数据验证（可选）
```

### 4. 启动桌面应用

#### 方式1：使用启动脚本（推荐）
```bash
start_app.bat
```

该脚本会自动：
- ✅ 检查Python环境
- ✅ 检查依赖安装
- ✅ 自动启动桌面应用

#### 方式2：手动启动
```bash
cd d:\测试开发项目\电商系统企业级+自动化框架\tianjin_bureau_system
python main.py
```

#### 方式3：打包后启动（可选）
```bash
# 使用PyInstaller打包
pyinstaller --onefile --windowed main.py

# 运行打包后的可执行文件
dist\main.exe
```

## 常见问题处理

### 问题1：无法启动Tkinter

**错误信息：**
```
ModuleNotFoundError: No module named 'tkinter'
```

**解决方案：**
```bash
# Windows
pip install tk

# Linux
sudo apt-get install python3-tk

# macOS
brew install python-tk
```

### 问题2：数据库连接失败

**错误信息：**
```
pymysql.err.OperationalError: (2003, "Can't connect to MySQL server")
```

**解决方案：**
```bash
# 1. 检查MySQL服务
net start MySQL80

# 2. 检查配置文件
# 编辑 config.py 中的 DB_CONFIG

# 3. 测试连接
mysql -h localhost -u root -p
```

### 问题3：窗口无法显示

**错误信息：**
```
_tkinter.TclError: no display name and no $DISPLAY environment variable
```

**解决方案：**
- Windows：不需要特殊处理
- Linux：需要安装X11服务器
- macOS：可能需要安装XQuartz

### 问题4：字体显示异常

**错误信息：**
```
字体乱码或方块
```

**解决方案：**
```python
# 在main.py中添加字体设置
import tkinter.font as tkfont
root = tk.Tk()
root.option_add("*Font", "Microsoft YaHei 10")
```

## 打包发布

### 使用PyInstaller打包

```bash
# 安装PyInstaller
pip install pyinstaller

# 打包为单个可执行文件
pyinstaller --onefile --windowed main.py

# 打包为文件夹（推荐）
pyinstaller --onedir --windowed main.py
```

### 打包配置文件

创建 `main.spec`:
```python
# -*- mode: python ; coding: utf-8 -*-
block_cipher = None

a = Analysis(['main.py'],
             pathex=['D:\\test\\tianjin_bureau_system'],
             binaries=[],
             datas=[('db', 'db'), ('config.py', '.')],
             hiddenimports=['pymysql', 'tkinter'],
             hookspath=[],
             runtime_hooks=[],
             excludes=[],
             win_no_prefer_redirects=False,
             win_private_assemblies=False,
             cipher=block_cipher,
             noarchive=False)

pyz = PYZ(a.pure, a.zipped_data,
             cipher=block_cipher)

exe = EXE(pyz,
          a.scripts,
          [],
          exclude_binaries=True,
          name='天津市市政工程局管理系统',
          debug=False,
          bootloader_ignore_signals=False,
          strip=False,
          upx=True,
          console=False,
          disable_windowed_traceback=False,
          argv_emulation=False,
          target_arch=None,
          codesign_identity=None,
          entitlements_file=None,
          icon=None)

coll = COLLECT(exe,
               a.binaries,
               a.zipfiles,
               a.datas,
               strip=False,
               upx=True,
               upx_exclude=[],
               name='天津市市政工程局管理系统',
               debug=False,
               bootloader_ignore_signals=False,
               strip=False,
               upx=True,
               upx_exclude=[],
               runtime_tmpdir=None)
```
